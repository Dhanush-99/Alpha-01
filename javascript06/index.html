Problem:

You are given a string (STR) of length N, consisting of only the lower case English alphabet.
Your task is to remove all the duplicate occurrences of characters in the string.

Input:
abcadeecfb

Output:
abcdef

Ans.

                let str = "abcadeecfb";
                let remove_duplicate = (str) => {
                    let setObject = new Set();
                    let string = "";
                    for (let i = 0; i < str.length; i++) {
                        setObject.add(str[i]);
                    }
                    for (const key of setObject.keys()) {
                        string += key;
                    }
                    return string;
                }
                console.log(remove_duplicate(str));

Problem:

You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps)

Input:
abcadeecfb

Output:
a=2
b=2
c=2
d=1
e=2
f=1

Ans.

                let s="abcadeecfb";
                let map = new Map();
                
                for(let i=0; i< s.length; i++){
                  if(map.has(s[i])){
                    let previousCount= map.get(s[i]);
                    map.set(s[i],previousCount+1);
                  }
                  else{
                    map.set(s[i], 1);
                  }
                }
                for(let [ch, freq] of map.entries()){
                  console.log(ch+"="+freq);
                }
